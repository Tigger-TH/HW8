HW7: ตั้งค่า IIS และพัฒนาเว็บเบื้องต้น

IIS · Port 8080 HTML5 CSS

งานชิ้นนี้เป็นการฝึกตั้งค่าเว็บเซิร์ฟเวอร์ Internet Information Services (IIS) บนเครื่องส่วนตัวให้รันที่พอร์ต 8080 (http://localhost:8080) พร้อมพัฒนาเว็บหน้าเดียวสำหรับแนะนำตัว แสดงชื่อ รหัสนักศึกษา สาขา และสถาบัน โดยใช้ HTML ควบคู่กับไฟล์ CSS ภายนอกเพื่อจัดรูปแบบ

เว็บหน้าเดียวนี้เองที่กลายมาเป็นจุดตั้งต้นของเว็บไซต์ส่วนตัวฉบับเต็มใน HW8 — จากการ์ดโปรไฟล์หนึ่งใบ ขยายเป็นเว็บไซต์หลายหน้าที่มี Home, About, CV, Portfolio และ Contact ครบถ้วน

สิ่งที่ได้เรียนรู้

  • การติดตั้งและตั้งค่า IIS ให้รันเว็บไซต์ Static บนพอร์ตที่กำหนด
  • โครงสร้างพื้นฐานของ HTML5 และการเชื่อมไฟล์ CSS ภายนอก
  • การจัดวางข้อมูลส่วนตัวให้อ่านง่ายด้วย Card Layout และเงา (Box Shadow)
